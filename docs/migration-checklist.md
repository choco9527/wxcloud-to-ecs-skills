# 通用验证与回滚清单

## 一、迁移前检查

- 已盘点所有云函数入口
- 已盘点数据存储使用面
- 已盘点文件存储使用面
- 已盘点后台任务/定时触发器
- 已确认前端请求入口和环境判断方式

## 二、后端服务验证

- 服务可正常启动
- 健康检查可访问
- 鉴权链路可用
- 核心接口返回结构符合预期
- 日志中无明显启动错误

## 三、数据链路验证

- 关键数据可读
- 关键数据可写
- 历史数据兼容策略明确
- 批量导入具备幂等或可重复执行能力

## 四、文件链路验证

- 上传接口可用
- 返回的资源 URL 可访问
- 历史资源仍可读取
- 本地调试与线上资源路径策略明确

## 五、前端联调验证

- 请求层已切换到统一入口
- 本地环境可调试
- 非本地环境不会误连本地服务
- 核心页面无白屏、无结构不兼容错误

## 六、后台任务验证

- 定时任务可触发
- 手动触发方式明确
- 任务日志可查看
- 失败可重试

## 七、发布前检查

- 域名与 HTTPS 条件满足
- 合法业务域名已配置
- 灰度范围明确
- 回滚步骤可执行

## 八、回滚检查

- 旧链路仍可临时启用
- 前端切回方式明确
- 后端可快速回退到上一版本
- 数据迁移过程中无不可逆破坏步骤
